Viewing Word Variations

Choose Word from the main menu to view many different variations of words. This can be very handy for quickly finding hooks, anagrams, or other variations. The variations in the Word menu are also available by right-clicking any word in any word table. The available variations are described below.

Word Definition

If definitions were given in the currently loaded lexicon file (see Loading a Lexicon), then choosing Definition will allow you to see a word's definition. If no definitions were loaded, it will still show whether the word is acceptable or unacceptable.

Anagrams

Choosing Anagrams will display all valid words that use the same letters as the specified word, with the letters in any order. For example, Anagrams of SATIRE are AIREST, SATIRE, STRIAE, and TERAIS.

Subanagrams

Choosing Subanagrams will display all valid words that can be formed using the letters in the specified word. For example, Subanagrams of RUST are US, UT, RUT, UTS, RUST, and RUTS.

Hooks

Choosing Hooks will display all valid words that can be formed by adding a letter to either end of the specified word. For example, Hooks of HOOK are CHOOK, SHOOK, HOOKA, HOOKS, and HOOKY.

Extensions

Choosing Extensions will display all valid words that can be formed by adding any number of letters to either end of the specified word. For example, Extensions of RHUS are SCIRRHUS and RHUSES.

Anagram Hooks

Choosing Anagram Hooks will display all valid words that can be formed by adding a single letter to the specified word, and rearranging the letters in any order. For example, Anagram Hooks of BATH are BAHTS, BAITH, BATCH, BATHE, BATHS, and HABIT.

Blank Anagrams

Choosing Blank Anagrams will display all valid words that can be formed by changing a single letter in the specified word, and rearranging the letters in any order. For example, Blank Anagrams of CZAREVNA are ADVANCER, CANVASER, CZAREVNA, TZAREVNA, and VARIANCE.

Blank Matches

Choosing Blank Matches will display all valid words that can be formed by changing a single letter in the specified word, and keeping the letters in the same order. For example, Blank Matches of TRAILER are FRAILER, TRAILED, TRAILER, TRAINER, and TRAWLER.

Transpositions

Choosing Transpositions will display all valid words that can be formed by swapping two adjacent letters in the specified word. For example, a Transposition of STRONGLY is STRONGYL.
